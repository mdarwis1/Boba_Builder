# Boba Builder

## Overview
Mobile application that allows users to customize their own boba drinks, save drinks as favorites, and simulate paying for drinks at checkout.

## Features
- Simple login gatekeeper 
- Custom drink builder with dynamic images
- Various UI features (dropdown, slider, button, etc.) for customization
- Favorite screen with drinks list
- Rename, delete, and add to cart functions for favorites
- Preset drink option
- Checkout screen with drinks in cart
- Drink description modal when item clicked
- Delete function for cart items
- Calculated tip options

## Technologies Used
- React Native
- Expo Go
- TypeScript
- SQLite

## What I Learned
- Building a full React Native app with multiple screens and tab navigation
- Managing a local database with SQLite
- Accessing drink data across screens
- Creating edit and delete functions for drinks using SQLite
- Dynamically switching images based on user selection
- Implementing UI such as dropdowns and sliders
