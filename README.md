# Boba Builder

## Overview
Mobile application that allows users to customize their own boba drinks, save drinks as favorites, and simulate paying for drinks at checkout with a shopping cart system.

## Features

### Account
- Gatekeeper for user login

### Builder Screen 
- Custom drink builder with dynamic images
- Adjustable drink options (size, base, flavor, sweetness, boba)

### Favorites Screen
- Favorites system to save, rename, and delete drinks
- Add favorite drinks to cart
- Preset drink option
  
### Checkout Screen
- Checkout system to save and delete drinks in cart
- Drink detail modal when item clicked
- Tip selection with total price calculation 

## Technologies Used
- React Native
- Expo Go
- TypeScript
- SQLite

## What I Learned
- Building a full React Native app with multiple screens and tab navigation
- Managing a local database with SQLite
- Accessing drink data across screens
- Creating edit and delete functions for drinks using SQLite
- Dynamically switching images based on user selection
- Implementing UI such as dropdowns and sliders
